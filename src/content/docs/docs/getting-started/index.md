# Welcome to Aidxn Design

Aidxn Design is a solo full-stack web design agency run by Aiden Wood, specialising in high-performance websites, branding, and digital marketing for businesses across Australia.

## What We Do

We build fast, modern websites and digital experiences using cutting-edge technology. Every project is crafted with performance, accessibility, and conversion in mind.

- **Web Design & Development** — Custom Astro, React, and Next.js websites built for speed
- **Branding & Identity** — Logo design, brand guidelines, and visual identity systems
- **Digital Marketing** — Google Ads management, SEO, and analytics dashboards

## Why Aidxn Design?

- **One point of contact** — You work directly with the builder, not through layers of account managers
- **Full-stack capability** — From design to deployment to analytics, every layer is handled in-house
- **Performance-first** — Sites built on Astro achieve near-perfect Lighthouse scores out of the box
- **Transparent process** — Clear timelines, honest communication, no surprises
