## Overview

The client dashboard is your central hub for tracking project progress, reviewing deliverables, and communicating with the team. It provides real-time visibility into where your project stands at every stage.

:::note
The client dashboard is currently in development. This page will be updated with full documentation when the dashboard launches.
:::

## Planned Features

### Project Timeline
A visual timeline showing each phase of your project — discovery, design, development, and deployment — with status indicators and estimated completion dates.

### Deliverable Review
A dedicated space for reviewing design mockups, staging sites, and other deliverables. Leave comments and approve work directly from the dashboard.

### Communication
Centralised messaging so all project communication lives in one place. No more digging through email threads.

### File Sharing
Upload brand assets, content documents, and reference materials. Download completed deliverables and final files.

### Invoicing
View and pay invoices directly from the dashboard. Track payment history and upcoming milestones.

## Access

Once your project begins, you will receive login credentials for your dashboard. Each dashboard is unique to your project and accessible from any device.
